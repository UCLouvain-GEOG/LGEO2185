# LGEO2185
Advanced geo-processing (LGEO2185)

This repository contains course materials and resources for the LGEO2185 Advanced Geo-Processing course from the School of Geography (UClouvain). The course focuses on advanced techniques and tools for processing and analyzing geospatial data.

See website with course material: [LGEO2185 Course Page](https://uclouvain-geog.github.io/LGEO2185/)
